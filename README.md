Griffin.Container
=================

Inversion of control container with (almost) zero configuration.

also knows as "The drunken container".

# Core

## Installation

Install using nuget:

`install-package griffin.container`

## Documentation

[MSDN style docs](http://griffinframework.net/docs/container/)
